My solutions for [Advent of Code 2025](https://adventofcode.com/2025), written in [Zig](https://ziglang.org/).

## Running Solutions

You can run the solutions using the Zig build system.

### Run Everything

Executes all available solutions in sequential order.

```bash
zig build run
```

### Filter by Day

Run both parts for a specific day (e.g., Day 1).

```bash
zig build run -Dday=1
```

### Filter by Part

Run a specific part across all days (e.g., only Part 1s).

```bash
zig build run -Dpart=1
```

### Run a Specific Solution

Target a specific challenge (e.g., Day 1, Part 1).

```bash
zig build run -Dday=1 -Dpart=1
```
